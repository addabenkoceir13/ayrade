## How To Use

### Download Repository
> clone repository
```bash
 git https://github.com/addabenkoceir13/ayrade
```
> open folder
```bash
 cd ayrade
```
> Create file .env
```bash
 cp .env.example .env.
```
> Generate Key Of .env
```bash
 php artisan key:generate.
```

### Create DataBase && Migration && Seeding
> Create DataBase
```bash
 CREATE DATABASE IF NOT EXISTS 'ayrade'
```
> Go to file .env
```bash
DB_DATABASE=ayrade
```
# Migration Table and seeder
```bash
 php artisan migrate --seed

```
